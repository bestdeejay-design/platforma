# Сайт АО «Платформа» · цифровая инфраструктура

## Описание проекта

Сайт представляет собой веб-платформу для цифровой инфраструктуры локальной экономики. Проект реализован как современный одностраничный сайт с адаптивным дизайном, темной и светлой темами, и системой навигации.

## Текущие особенности сайта

1. **Адрес компании**: г. Санкт-Петербург, Таврическая улица, 7 (обновлено из предыдущего "Измайловский 4")
2. **Структура файлов**:
   - `index.html` - основная HTML-страница
   - `styles.css` - стили оформления
   - `script.js` - JavaScript функциональность
3. **Навигационные кнопки**:
   - Появляются после секции "Продукты и решения"
   - Кнопки со стрелками вверх/вниз для навигации между секциями
   - Реализованы с использованием Font Awesome иконок
4. **Темная/светлая тема**:
   - Переключение темы сохраняется в localStorage
   - Полностью функциональная темная тема по умолчанию
   - Светлая тема с градиентами и эффектами

## Функциональность

- Адаптивный дизайн для различных устройств
- Плавная прокрутка к секциям
- Интерактивные элементы с hover-эффектами
- Система карточек с товарами и услугами
- Темизация интерфейса

## Технологии

- HTML5
- CSS3 (с переменными и адаптивной версткой)
- JavaScript (модульная архитектура)
- Font Awesome (иконки)

- Герой-секция с основным предложением
- Статистика компании

- Партнеры
